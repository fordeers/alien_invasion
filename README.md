Work in progress..

Arrow Keys - For ship movement 

Spacebar - For firing

Escape - to pause or go back on main menu

D - For upfront triple bullets

S - For side bullets 

R - To reset stats and restart from level 1

F - Fulscreen game / Exit Fullscreen

Q - to quit game

Player start with 3 lives.
Points increase for each Alien ship hit as you progress every round.

Level 0 - game/base speed 

Level 1 - game/base speed x2

Level 2 - game/base speed x3

Level 3 - game/base speed x4

Level 4 - game/base speed x5

Level 5 - game/base speed x6

and so on and so forth...
